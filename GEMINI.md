# Project Organization

Adhere to the following directory structure for all files:

- **Raw Data:** Save freshly downloaded or raw datasets in the `data/` folder.
- **Processed Data:** Save cleaned or processed datasets in the `data/clean/` folder.
- **Scripts:** Save all code and execution scripts in the `scripts/` folder.
- **Maps & Visuals:** Save generated maps and visualizations in the `maps/` folder.
- **Logs:** Save execution logs and diagnostic output in the `logs/` folder.
